PyBursa Description

Описание курса PyBursa

На курсе мы Вас познакомим с фреимворком Djangot, вместе сделаем работающий проект на Python / Django и освоим еще много полезного.

Program

Программа курса PyBursa

  • Первая неделя. Основы html/css. GIT

    1. html - теги и свойства
    2. css - свойства и селекторы
    3. Bootstrap
    4. GIT basic reference

  • Вторая неделя. Django first steps

    1. введение и описание фреймворка, старт проекта
    2. инструменты отладки браузера
    3. HTTP (GET, POST), (200, 404, 500), headers, cookies
    4. админка

  • Третья неделя. Структура Web приложения. Работа с базами данных, ORM

    1. MVC
    2. Views
    3. роутинг урлов, передача параметров
    4. обработка реквеста, формирование респонса
    5. мидлвар
    6. модели и поля, типы полей и аттрибуты
    7. связи между таблицами и проектирование БД
    8. миграции
    9. QuerySet

  • Четвертая неделя. Кастомизация админки. Язык шаблонов

    1. настройка отображения виджетов админки
    2. сортировка, фильтры, поиск
    3. синтаксис шаблонов, логические конструкции
    4. наследование и includes шаблонов
    5. теги и фильтры

  • Пятая неделя. Forms. Class-based (Generic) Views.

    1. формы, типы полей, виджеты и аттрибуты, валидация
    2. встроеные class-based views (TemplateView, ListView, ItemView)
    3. синтаксис шаблонов, логические конструкции
    4. миксины

  • Шестая неделя. Реализация стандартных задач. Средства отладки и тестирование. Деплоймент.

    1. отправка емейл, кеширование, интернационализация, батарейки (готовые решения)
    2. logging, pdb, debug toolbar, extensions, unit-tests, selenium
    3. nginx, управление статикой, деплоймент
